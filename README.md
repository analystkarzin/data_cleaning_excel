# Excel Data Cleaning Project

## Overview
This project demonstrates data cleaning techniques using Microsoft Excel.
The dataset was sourced from a public YouTube tutorial and cleaned independently.

## Tools Used
- Microsoft Excel (365)
- Filters & sorting
- IF / IFERROR formulas
- Data validation

## Data Cleaning Steps
- Identified and filtered blank cells
- Removed unnecessary rows
- Handled #DIV/0! errors using IFERROR
- Standardized numeric and text formats
- Verified data consistency

## Outcome
- Produced a clean, analysis-ready dataset
- Improved data accuracy and usability
