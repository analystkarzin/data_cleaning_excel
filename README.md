# Excel Data Cleaning Project

## Overview
Performed data cleaning and preparation using Excel techniques including error handling, duplicate removal, formatting standardization, and quality checks to prepare datasets for analysis.

## Tools Used
- Microsoft Excel (365)
- Filters & sorting
- IF / IFERROR formulas
- Data validation

## Data Cleaning Steps
- Identified and filtered blank cells
- Removed unnecessary rows
- Handled #DIV/0! errors using IFERROR
- Standardized numeric and text formats
- Verified data consistency

## Outcome
- Produced a clean, analysis-ready dataset
- Improved data accuracy and usability
